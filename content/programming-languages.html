Programming Languages

1950s–1960s

  • Fortran (1957): numeric computing and scientific work.
  • Lisp (1958): symbolic processing and AI research.
  • COBOL (1959): business data processing.
  • ALGOL (1958/60): structured programming concepts.

1970s–1980s

  • C (1972): systems programming and portability.
  • Smalltalk (1972): objects, messages, and modern IDE ideas.
  • SQL (1974): relational databases and declarative queries.
  • C++ (1985): object-oriented features on top of C.

1990s–2000s

  • Python (1991): readability and batteries-included.
  • Java (1995): write once, run anywhere (JVM).
  • JavaScript (1995): dynamic web pages in the browser.
  • PHP (1995) and Ruby (1995): rapid web development.

2010s–Today

  • Go (2009): simplicity and concurrency.
  • Rust (2010): safety without garbage collection.
  • TypeScript (2012): typed superset of JavaScript.
  • Kotlin/Swift: modern language design for apps.
